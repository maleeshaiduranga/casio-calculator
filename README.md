# SciCalc Pro - Android App

A complete, native Android application that replicates the layout of a classic scientific calculator. Built using Kotlin and Jetpack Compose.

## Download App

Click the button below to download the latest Android APK directly:

[![Download APK](https://img.shields.io/badge/Download-APK-green?style=for-the-badge&logo=android)](https://github.com/maleeshaiduranga/casio-calculator/releases/download/latest/app-debug.apk)

*(Make sure to enable "Install from Unknown Sources" on your device if prompted)*

## Features
- High-fidelity classic scientific calculator layout and UI.
- Dual-line display for input formulas and results.
- Supports SHIFT and ALPHA states.
- Powered by `mXparser` for advanced mathematical evaluations.
